# TABERTA

TABERTA is a family of structure-aware sentence-transformer models for table
retrieval, table discovery, and retrieval-augmented table question answering.

This repository hosts public TABERTA resources for both inference and
fine-tuning.

## Contents

- `TABERTA_Inference/` - model download, dataset preparation, embedding
  generation, vector indexing, and evaluation scripts.
- `TABERTA_7_Finetuning_Strategies/` - training code for the seven TABERTA
  fine-tuning strategies.

Large model files, dataset archives, generated embeddings, vector indexes, and
local database storage are intentionally not tracked in Git. Each package
contains its own setup and usage guide.
